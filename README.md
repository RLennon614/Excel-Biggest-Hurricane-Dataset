# Excel-Biggest-Hurricane-Dataset
Overview
This repository contains a data analysis project focused on the Biggest Atlantic Hurricanes based on historical data. The project utilizes Excel to perform sorting, ranking, calculations, and data visualization to derive insights into the strongest and most impactful tropical cyclones.

This serves as a practical demonstration of advanced spreadsheet skills in data cleaning, analysis, and presentation.
